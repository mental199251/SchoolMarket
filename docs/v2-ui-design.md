# V2 UI 设计说明

V2 将 V1 的功能链路保持不变，重点重写前端视觉系统与交互质感，覆盖 H5 和微信小程序。

## 1. 设计目标

- 美观、可爱、轻盈，有校园二手交易的亲近感。
- 保持主要功能不变：注册登录、商品、交易、消息、管理端、统计和 AI 发布助手。
- H5 宽屏有更舒展的内容宽度，小程序端保持触控友好。
- 动效使用轻微呼吸、浮动、弹性按压和卡片入场，不阻塞操作。

## 2. 视觉系统

- 风格：soft clay / candy card / breathable marketplace。
- 主色：薄荷绿，用于主按钮、状态成功和关键行动。
- 辅色：奶油白、蜜桃橙、天空蓝，用于背景层次和柔和高光。
- 卡片：大圆角、双层阴影、内高光，形成柔软的触感。
- 字体：系统字体优先，保证 H5 和小程序一致性与加载稳定。

## 3. 交互与动效

- 页面背景使用低频呼吸动画。
- 首页数字卡片使用轻微浮动。
- 卡片和按钮提供按压反馈。
- 列表、表单、统计条形图使用短时入场动画。
- 支持 `prefers-reduced-motion`，减少动画偏好下关闭主要动效。

## 4. 响应式适配

- 移动端优先，使用 `rpx` 保持小程序适配。
- H5 宽屏内容居中，最大宽度控制在 980px。
- 商品列表在宽屏下切换为双列卡片，在移动端保持单列。
- 按钮高度和表单输入高度满足触控尺寸要求。

## 5. 功能保护

V2 只重构 UI 层和视觉交互，未修改后端接口契约和业务状态机。已验证：

- 后端自动化测试通过。
- H5 构建通过。
- 微信小程序构建通过。
- 首页、商品列表、商品详情和发布页在 H5 中完成视觉检查。
