# 开发环境

## 版本

- Node.js：20，见 `.nvmrc`
- Python：3.11，兼容 3.12，见 `.python-version`
- MongoDB：7.x

## 端口

| 服务 | 默认端口 |
| --- | --- |
| uni-app H5 | 5173 |
| Flask API | 5001 |
| MongoDB | 27017 |
| Ollama | 11434 |

## 环境变量

复制 `.env.example` 为 `.env`。`.env` 不得提交到版本库。

开发环境允许 Flask 提供默认值，生产环境必须显式设置密钥、数据库地址、上传目录和跨域来源。

认证相关变量：

| 变量 | 默认值 | 说明 |
| --- | --- | --- |
| `JWT_SECRET_KEY` | `dev-school-market-secret` | JWT 签名密钥，生产环境必须替换 |
| `JWT_EXPIRES_SECONDS` | `604800` | 登录 token 有效期，默认 7 天 |
| `PASSWORD_MIN_LENGTH` | `8` | 注册和改密的最短密码长度 |

## 网络说明

- H5 可使用 `http://127.0.0.1:5001`。
- 微信开发者工具可在开发阶段关闭合法域名校验。
- 真机不能访问电脑的 `127.0.0.1`，需要使用电脑局域网 IP。
- Flask 开发服务监听 `0.0.0.0` 仅用于本地联调，不作为生产部署方式。

## 协作规则

- 每个模块遵循“接口契约、后端测试、后端实现、前端页面、联调、验收”的顺序。
- API、错误码或枚举变化必须与文档在同一提交中更新。
- 页面不得散落后端地址和接口路径。
- 路由层不实现复杂业务规则，数据库访问集中到 repository，状态机集中到 service。
